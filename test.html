FfadeToSrc fades between images using CSS3.

Get it

Download .zip Download .tar.gz View on GitHub

Usage

Fade an image to another image using CSS3 transitions, but invoked via JavaScript.

$('img').fadeToSrc("http://layervault.com/new_source.png");

You can also give it a few options:

$('img').fadeToSrc("New Subheading", {
  fadeInLength: 1000,
  fadeOutLength: 2000,
  fadeInEasing: 'ease-in',
  fadeOutEasing: 'ease-out'
});

Or event a callback:

$('img').fadeToSrc("New Subheading", function () {
  alert("Fade complete!");
});

Installation

Just include jquery.image.fadeto.js as a script tag on the page. Bonus points for incorporating it into your asset pipeline.
